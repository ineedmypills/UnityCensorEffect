# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [5.0.0] - 2025-08-16

### Added
- This is a brand new version, completely rewritten from scratch.

### Changed
- The entire codebase has been rewritten for clarity, maintainability, and robustness.
- The core logic still uses the reliable secondary camera approach for mask rendering, but the implementation is new.
- All shaders have been rewritten from scratch with improved commenting and structure.

### Removed
- All old code from previous versions.
